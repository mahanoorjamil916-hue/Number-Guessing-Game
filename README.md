# 🎮 Number Guessing Game (Python)

An interactive, object-oriented Command Line Interface (CLI) game built using Python.

### ✨ Key Features
* **OOP Logic:** Uses classes and methods for clean, organized code.
* **Error Handling:** Validates user input to prevent crashes.
* **Attempt Tracking:** Counts how many tries it takes to win.
* **Replayability:** Option to play again without restarting the script.

### 🛠️ Built With
* Python 3
* `random` module
