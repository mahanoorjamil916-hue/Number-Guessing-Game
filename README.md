# 🎮 Advanced Level-Based Number Guessing Game (Python)

Welcome to my **Syntexhub Internship Project #1**! This is a professional-grade Number Guessing Game built with Python, featuring multiple difficulty levels, robust error handling, and Object-Oriented Programming (OOP) principles.

## ✨ Key Features
* **Difficulty Levels:** Choose between **Easy, Medium, and Hard** levels, each with different number ranges and attempt limits.
* **Object-Oriented Design:** The entire game logic is encapsulated within the `NumberGuessingGame` class for clean and modular code.
* **User Experience (UX):** Integrated with emojis and clear feedback (Too High / Too Low) to make the game interactive.
* **Score Tracking:** The system tracks your **Best Score** across multiple rounds.
* **Input Validation:** Robust `try-except` blocks to handle invalid (non-numeric) inputs without crashing the game.
* **Replay System:** Allows players to restart the game or change levels immediately after finishing a round.

## 🛠️ Technical Implementation
* **Language:** Python 3
* **Modules:** `random` (for secret number generation)
* **Concepts Used:** Classes & Objects, Dictionaries (for levels), While Loops, Exception Handling, and F-strings.

## 🚀 How to Play
1. Run the script: `python main.py`
2. Enter your name to personalize the experience.
3. Select your difficulty level (easy/medium/hard).
4. Start guessing! Try to beat your best score in fewer attempts.
